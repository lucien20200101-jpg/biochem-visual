# biochem-visual

Biochemical Pathway Explorer（基于 Vite + React + React Router）的生化通路可视化小站，部署在 GitHub Pages。

## 在线访问（GitHub Pages）

- 主页：<https://lucien20200101-jpg.github.io/biochem-visual/#/>
- Map 页面：<https://lucien20200101-jpg.github.io/biochem-visual/#/map>

> 本项目使用 HashRouter，因此页面路径会以 `#/` 开头，这是 GitHub Pages 上最稳妥的路由方式。

## 本地运行（开发）

> 前提：已安装 Node.js（建议 18+ 或 20）

```bash
npm install
npm run dev
